# MyAppFlow — Landing page salon

Landing page à destination des prospects rencontrés en salon (accès via QR code).
Site statique (HTML / CSS / JS), sans dépendance de build, prêt à déployer sur Vercel.

## Contenu

- Mission
- Méthode en 5 étapes + bloc architecture (stack par défaut Claude Code / GitHub / Vercel, ou stack personnalisée)
- 3 projets en cours de développement : MyTelencia, MyIgiStudio, MySchool
- À propos
- Contact

## Développement local

Aucune installation requise : ouvrir `index.html` dans un navigateur, ou servir le dossier avec un serveur statique quelconque (ex. `npx serve .`).

## Déploiement

Le site est un site statique pur : Vercel le détecte automatiquement (aucune configuration de build nécessaire).
